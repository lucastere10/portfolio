Project IV: Womans Clothing Project

September 13, 2021

In this project we’ll use an e-commerce dataset from a womens’s clothing store. The dataset constitute of some client information, a titled review with rating and some product intel. My interest here is to analyze the costumer profile, product performace and make a Sentiment Analysis with TextBlox. As an extra, I will also make a simple Word Cloud.

Complete code is available at: Link 

        key features: EDA, Data Cleaning, Data Visualization, Sentiment Analysis, NLP, Word Cloud

Objectives:

  • EDA
  • Analyse customer preferences
  • Analyse for correlation between product features and customer preferences
  • Define what makes a good product
  • Define what makes product get a good review
  • Create a simple word count

Phases:

  • Data Preparation
  • Data Analysis and Vizualization
  • NLP and WordCloud

Data Preparation

Same as always, drop unused columns, fill NaNs, remove outliers. The null values were mostly in the title row, with some minor blank reviews. Change the NaN titles to ‘No Title’ and the NaN reviews to ‘Blank Review’.

Most NaN values on Division/Departament/Class were related too… socks? We also have at least 3 hoodies entries and a completely null review at index 13787. I droped the null review and after a better look at the features names found suitable categories for those items.

Lets put:

  • ID 72/152/184/136 as General/Bottoms/Socks
  • ID 492/772 as General/Jackets/Hoodies
  • ID 665 unfortunately can’t figure out, so it will be droped
